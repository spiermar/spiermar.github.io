- shared [agents.md](http://agents.md) or claude.md
- AI pr reviewer
- codeowners for PR review
- limit on size of PR
- doc with guidelines
    - need to understand the code that is being changed
    - if don’t know the codebase or don’t understand, reach out to owner
    - need to plan before execute
    - if planning a large change, discuss it with owner before
    - instruct to not change outside what was asked (AI slop)
    - don’t refactor
