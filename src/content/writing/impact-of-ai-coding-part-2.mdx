- everyone expected developer jobs to go away
- takes a lot less effort to build things
- you need less developers, job would go away
- the opposite started happening, demand increased for developers
- [The 2026 Global Intelligence Crisis](https://www.citadelsecurities.com/news-and-insights/2026-global-intelligence-crisis/)
- since it's cheaper to build, everyone wants to build more
- and more is being built
- the problem is the illusion of building fast and in large volumes is equal to improving things
- a lot can be built really fast, but not improve a product or innovate, just bloat
- the problem is described in the following blog post
- [Shipping Fast And Iterating At AI-Speed | Takeaways For Founders And Product Leaders](https://labs.adaline.ai/p/shipping-fast-and-iterating-at-ai)
- I cought myself multiple times developing new projects and feature for no reason, other than I felt like doing it
- things that would be used only once to try
- or that would be abandoned shortly after, when I lost interest
- previously, when the development bar was higher, it forced a priorization
- you had to choose carefully what to do
- and think about the reasons to do so
- now you can get up and running with minimal effort
- and paralellize development on multiple projects at the same time
- paradise for ADHD people like me
- development also requires a lot less friction
- switching between coding, running tests, reading docs, researching, troubleshooting
- now it's all a single flow of instructions, a single session
- so it's a very pleasant experience
- proven that by my garmin watch stress levels
- when I'm working with AI coding agents, my stress levels go to zero, for hours and hours
- flow state
- that was hard to achieve in the past, with all the context switching
- it's all great, but if we're developer a lot more, many times without actual improvements to a product, in a short amount of time, we will end up with a lot of abandoned code
- code that will likely need maintenance if it's running
- requiring even more developers to wrangle AI development agents
- a never ending snowball of job security
